  2. 眼部失用：无法随意眼动
  3. 视觉共济性失调

注意可以增强视觉加工区域间的神经震荡的想干性
背测注意网络主要负责注意定向，是双边的

返回抑制
extinction（对消）
对消：在忽视病人中，当向脑损伤同侧和对侧分别呈现一个刺激时，病人不能对脑损伤对侧刺激进行知觉或反应的现象。
IOR
参见返回抑制（inhibition of return）。
TRN
参见丘脑网状核（thalamic reticular nucleus）。
arousal / 唤醒
对机体的整体生理和心理状态的描述。其范围从深度睡眠到高度警觉。比较选择性注意 (selective attention)。
covert attention / 内隐注意
指在外部感受器没有变化的情况下产生的注意。例如，不动眼睛和头部而把注意转向说话者。比较外显注意 (overt attention)。
exogenous attention/外源性注意
参见反射性注意（reflexive attention）。
feature integration theory of attention /注意的特征整合理论
指视觉系统可以并行加工基本特征（如颜色、形状和运动），但需要空间注意来绑定所定义物体的各种特征的视知觉理论。
inhibition of return (IOR) /返回抑制
也称抑制性后效（inhibitory aftereffect），指在外源线索空间注意任务中观察到的一种现象。当注意被一个外源线索反射性地吸引到一个位置时，针对该线索出现300毫秒后所呈现刺激的反应会变慢。这就是返回抑制。

对原先注意过的物体或位置进行反应时所表现出的滞后现象
inhibitory aftereffect /抑制性后效
参见返回抑制（inhibition of return）。
late selection /晚选择
注意晚期模型认为，所有信息在知觉阶段都得到同等加工，而在晚期加工阶段注意对输入信息进行过滤。比较早选择（early selection）。

过滤器不在于选择知觉刺激，而在于选择对刺激的反应；所有输入的信息在进入过滤装置或衰减装置之前已经得到了充分的分析，然后再进入过滤或衰减装置，选择是发生在加工后期的反应阶段；选择标准是刺激对人
reflexive cuing /反射性线索
也称外源性线索（exogenous cuing）。一种在没有自主控制的情况下，使用外部感官刺激（如闪光）来自动吸引注意力的实验方法。比较内源性线索（endogenous cuing）。
selective attention /选择性注意
将注意力集中在某一感觉输入、思维或行为上，同时忽略其他因素的能力。应区分选择性注意与非选择性注意，后者包括简单的行为唤醒（选择性注意通常需要更多注意力）。比较唤醒（arousal）。
ventral attention network /腹侧注意网络
一种注意控制网络，涉及右腹侧额叶和颞顶联合区，基于刺激的新异性和突出性来调节反射性注意。比较背侧注意网络（dorsal attention network）。
voluntary attention /有意注意
也称内源性注意（endogenous attention），主动或有意将注意集中在某个输入源、想法或动作上。比较反射性注意（reflexive attention）。

随意注意又称有意注意，指有预定目的、需要一定意志努力的注意。引起随意注意的因素包括活动目的与任务（对目的与任务的理解、达到目的与完成任务的愿望）、兴趣（主要指间接兴趣）、活动组织、人格和过去的经验等
early selection / 早选择
在完成知觉分析和信息编码之前，注意在早期阶段就可以衰减或过滤感觉输入的注意模型。比较晚选择 (late selection)。

该理论认为，人的神经系统加工信息的容量是有限的，不可能对所有的感觉信息进行加工，这样就需要一个过滤器对信息进行选择，选择一些信息进入高级分析阶段，其他信息则被完全阻断在外。过滤器工作方式是“全或无”的。
neglect /忽视
参见单侧空间忽视（unilateral spatial neglect）。
overt attention /外显注意
将头部转向视觉、听觉、嗅觉或其他感官刺激。比较内隐注意（covert attention）。
bottleneck / 瓶颈
不是所有信息均可通过或进入的信息加工阶段。
endogenous cuing / 内源性线索
根据任务要求，使用符号提示 (如箭头) 诱导或指示被试自发地进行注意。比较反射性线索 (reflexive cuing)。
pulvinar /枕核
丘脑后部的一大片区域，由许多与皮质特定区域有双向连接的核团组成。
reflexive attention /反射性注意
也称外源性注意（exogenous attention）。由自下而上或数据驱动刺激引发的注意自动定向。例如，外周视野的一个闪光就会捕获你的注意。比较有意注意（voluntary attention）。
thalamic reticular nucleus (TRN) /丘脑网状核
围绕丘脑核团周围的一层薄薄的神经元，接收来自皮质和皮质下结构的输入，并向丘脑中转核团发送信号。
unilateral spatial neglect /单侧空间忽视
也称忽视（neglect），指前脑损伤的病人对呈现于损伤半球对侧的物体或事件无法检测或检测迟缓的行为模式，通常与右侧顶叶损伤有关。
visual search /视觉搜索
在多种刺激的画面或场景中寻找特定刺激的视觉任务。
dorsal attention network / 背侧注意网络
研究者提出的一种注意控制网络，包括背侧额叶和顶叶皮质，调节有意注意。比较腹侧注意网络 (ventral attention network)。
exogenous cuing/外源性线索
参见反射性线索（reflexive cuing）。
endogenous attention / 内源性注意
有意注意 (voluntary attention)。
